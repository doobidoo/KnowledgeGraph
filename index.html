Knowledge Graph Help

Explore the DokuWiki knowledge base as an interactive graph. Pages, tags, and namespaces are visualized as connected nodes.

Getting Started

Type a wiki page name in the search bar (e.g. start or intern:delivery:start) and press Go. Or click the shuffle button for a random page.

Node Types

Page — A wiki page. Shows its title and links to other pages.
Tag — A DokuWiki tag. The number in parentheses shows how many pages use this tag across the wiki.
Namespace — A namespace start page (red square). These represent the root of a wiki section.

Interactions

Click a node to expand it and reveal its linked pages and tags.

Double-click a page node to open it in the wiki. Double-click a tag to see all pages with that tag in the wiki.

Hover over a node to see its details and highlight the path back to the starting page.

Scroll to zoom in and out. Drag the background to pan.

URL Parameters

Share a specific view: ?page=intern:delivery:start

Embed mode (hides UI): ?page=start&embed=1
